What is Wine-o-meter ?

Wine-o-meter is a application that allows wine producers to predict the quality score of their wine based on physicochemical inputs. The data-science team developed a machine learning model able to predict the global quality of a wine. The score (from 0 for very bad, to 10 for excellent) is predict from attributes of the wine. You can easely request the Wine-o-meter API endpoint.

Model and input description

The model takes 11 inputs :
- fixed acidity
- volatile acidity
- citric acid
- residual sugar
- chlorides
- free sulfur dioxide
- total sulfur dioxide
- density
- pH
- sulphates
- alcohol
For more information please check the Model_Training.ipynb file in the repo https://github.com/valskyyy/5_deployment

API endpoint

You can access to the endpoint here (/predict).
This endpoint accepts POST method with JSON input data and it should return the predictions. We assume inputs will be always well formatted.

Method 1 : Curl

Example: curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://wine-o-meter-api.herokuapp.com/predict

Method 2 : Python

import requests
response = requests.post("https://wine-o-meter-api.herokuapp.com/predict", json={
"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
})
print(response.json())

Response

It returns a JSON like so:

{ 'the prediction is': [6] }
